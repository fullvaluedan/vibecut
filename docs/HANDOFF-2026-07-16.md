## Hard-won lessons THIS session (enforce on every subagent)

- NO em dashes anywhere, chat or code (Dan's standing rule; grep added lines for U+2014).
- Never edit files via PowerShell string replacement; single-pair arrays flatten and
  `.Replace(char,char)` corrupted three files (all recovered from HEAD). Use the Edit tool.
- The eval cache keys on pass INPUT only: any prompt WORDING change must bump the pass's
  prompt-version constant (see `VERIFY_PROMPT_VERSION`) or the eval silently replays stale
  responses. The recall passes do not have version constants yet; add one before revising
  their prompts.
- The structural pass's cache busts whenever retake toggles (its handledSpans include retake
  rows), so cross-arm comparisons re-roll the structural sample: single-run variance swung
  google 63.8 -> 58.8 on the same config family. Gate decisions need `--runs 3` aggregation.
- New DirectorOpCategory values need: taste.ts CATEGORY_LABEL AND the CATEGORIES array,
  review-format badge, and the `justify-cuts.ts` JUSTIFIED_REMOVAL allow-list.
- bun `mock.module` is process-global: every director route test must stub EVERY planner.
- Probe-then-tune: read the cached LLM response per confidence band / per position before
  spending live tokens (scratchpad probe scripts; the pattern found the 0.6 structural floor
  and the hermes whole-line-bleed evidence).
- Per-fixture eval runs: copy one fixture json into its own temp dir and pass the dir as the
  positional arg. Run foreground with generous timeouts; never `claude --version` in this repo;
  presence-check with `where claude`.
